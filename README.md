# Tic Tac Toe

A clean, minimal Tic-Tac-Toe web application for two human players on the same device.

## Features

- **Pass-and-Play**: Two players share the same device
- **Win Detection**: Detects wins in rows, columns, and diagonals
- **Draw Detection**: Recognizes when the board is full
- **Responsive Design**: Works on mobile and desktop
- **Clean UI**: Minimal, intuitive interface with no unnecessary elements

## How to Play

1. Open `index.html` in your browser
2. Player X goes first
3. Click any empty cell to place your mark
4. First player to get 3 in a row wins
5. Use the Reset button to start a new game

## Technologies

- Vanilla HTML5
- CSS3 (Responsive Grid Layout)
- JavaScript (No frameworks or libraries)

## Game Board

The 3×3 board is responsive and works on all screen sizes.
